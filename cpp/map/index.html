    std::map

    基本功能

    维护一棵pair<key,T>的二叉搜索树。在同一个map实例中key是唯一的，且key必须是可比较大小的类型。

    insert()方法

    向map中插入元素，若已经存在key相同元素则不插入。

    比较常用的重载介绍如下：

    • 重载一：
      • 参数是std::pair类型。需要使用make_pair()方法组合key和T，构造参数，再调用insert()方法。
      • 返回值是std::pair类型。pair.second是bool型，表示是否成功插入。如果成功插入，pair.first是指向被插入元素的迭代器。如果未成功插入，pair.first是指向key值与参数相同的那个元素的迭代器，即指向阻止插入进行的那个元素的迭代器。
    std::map<int, int> m;
auto p = m.insert(make_pair(2, 3));
std::cout << p.second << std::endl;   // true
std::cout << p.first->second << std::endl;
p = m.insert(make_pair(2, 5));
std::cout << p.second << std::endl;   // false
std::cout << p.first->second << std::endl; // m[5] = 3 here

    operator[]方法

    返回某个key对应元素的引用，若不存在该key对应的元素，则先插入该key对应的元素，再返回引用。

    比较常用的重载介绍如下：

    • 重载一：
      • 参数是key的左值引用常量。
      • 返回值是该key对应节点的T的引用。
    • 重载二：
      • 参数是key的右值引用。
      • 返回值是该key对应节点的T的引用。
    std::map<int, int> m;
m[2] = 1;
std::cout << m[2] << std::endl;
m[2] = 5;
std::cout << m[2] << std::endl;

    1
5

    erase()方法

    从map中删除某个指定的元素。

    比较常用的重载介绍如下：

    • 重载一：
      • 参数是指向被删元素的迭代器。
      • 返回值是删除对应元素之后，指向下一个元素的迭代器。
    • 重载二：
      • 参数是分别指向被删区间头尾的两个迭代器，左闭右开。
      • 返回值是删除对应区间之后，指向下一个元素的迭代器。
    • 重载三：
      • 参数是key值。
      • 返回值是成功删除的元素个数。
    #include <map>
#include <iostream>
int main()
{
    std::map<int, std::string> c = {
        {1, "one" }, {2, "two" }, {3, "three"},
        {4, "four"}, {5, "five"}, {6, "six"  }
    };

    // erase all odd numbers from c
    for(auto it = c.begin(); it != c.end(); ) {
        if(it->first % 2 != 0) {
            it = c.erase(it);  // Attention! See how to erase element here. Alternative: c.erase(it++);
            continue;
        }
        ++it;
    }

    for(auto& p : c) {
        std::cout << p.second << ' ';
    }
}

    two four six

    find()方法

    在map中查找是否存在给定key值的元素。

    比较常用的重载介绍如下：

    • 重载一：
      • 参数是给定key值的左值引用常量。
      • 若查找成功，返回指向对应元素的迭代器，若查找失败，返回对应map的end()迭代器。
    std::map<int, std::string> c = {
        {1, "one"}, {2, "two"}, {3, "three"}};
auto it = c.find(1);
if (it != c.end())
    std::cout << "success" << std::endl;
else
    std::cout << "fail" << std::endl;

    success
