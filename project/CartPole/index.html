    CartPole

    Introduction

    Use Actor-Critic, a kind of Policy Gradient Algorithm in RL. Thanks to RL An Introduction: 2nd Edition.

    Use predicted policy to explore and generate trajectories. Use predicted policy to calculate return value and learn. (on-policy)

    Use pytorch machine learning framework to implement the algorithm.

    Use openai gym CartPole-v1 as training environment. Every step you hold the pole, you get a reward of 1.

    Understanding

    \(\gamma\) is an important parameter for return value. Theoritically, if \(\gamma\)=0.9, an agent can get an expectation of \(1+0.9+0.9^2+\dots+0.9^n=\frac{1-0.9^{n+1}}{1-0.9}\)，nearly \(10\) reward. That means a normal state's value is nearly \(10\).

    In my program, I print the predicted value of states long time after training started. Then I find that a good state (far from failure) has a value about \(10\). \(\gamma=0.95\) is same condition.

    Actor-Critic algorithm use two statements to update parameter \(\theta\) and \(w\). (See note).

    Our goal is maximizing the probability of an action with higher value in a policy and making the value function apporaching true value of a state.

    Here, \(\delta=R_t+\gamma v(S_{t+1},w)-v(S_t,w)\) means how large the gap between a state's predicted value and a state's expected value (here return). If it is positive, \(ln\pi(A_t|S_t,\theta)\) and \(v(S_t,w)\) should become larger. If it is negative, they should become smaller. Gradient means the direction that function value increases most quickly, so we add the result of gradient of these two functions and \(\delta\) to \(\theta\) and \(w\) .

    For more information about this formula, See note.

    The norm of these two gradient is not very important because it only correlates with initial value and neural network's properties. After many iteration, the convergence will not be affected by this norm.
