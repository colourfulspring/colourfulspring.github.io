    Codeforces Round #843(Div.2) B Gardener and the Array

    思路

    题目要求判断给定一个数组\(c_1,c_2,\dots,c_n\)，是否存在两个不同的子序列\(a\)和\(b\)，使得\(f(a)=f(b)\)，\(f\)是序列元素按位或函数。其中\(c_i\)是通过给定所有被置位的bit位置编号来满足的。

    根据题意，可以发现：如果存在一个\(c_i\)，让\(c_1|\dots|c_{i-1}|c_{i+1}|\dots|c_n\)和\(c_1|\dots|c_n\)完全相同，则答案为YES。接下来只要证明，反过来也成立，即可得充要条件。

    要证明YES推出存在一个\(c_i\)，让\(c_1|\dots|c_{i-1}|c_{i+1}|\dots|c_n\)和\(c_1|\dots|c_n\)完全相同。可以证明逆否命题，不存在一个\(c_i\)，让\(c_1|\dots|c_{i-1}|c_{i+1}|\dots|c_n\)和\(c_1|\dots|c_n\)完全相同推出NO。

    不存在一个\(c_i\)，让\(c_1|\dots|c_{i-1}|c_{i+1}|\dots|c_n\)和\(c_1|\dots|c_n\)完全相同等价于任意\(c_i\)，\(c_1|\dots|c_{i-1}|c_{i+1}|\dots|c_n\)和\(c_1|\dots|c_n\)不同。这说明少了任意一个数都会发生无法用其他数弥补的变化，显然一定是NO了。

    综上，YES的充要条件是存在一个\(c_i\)，让\(c_1|\dots|c_{i-1}|c_{i+1}|\dots|c_n\)和\(c_1|\dots|c_n\)完全相同。

    P.S. 这整个题的结构与线性代数中的极大线性无关组很像。我们可以仿照线性代数推出相似的或代数。定义或无关是任意\(c_i\)，\(c_1|\dots|c_{i-1}|c_{i+1}|\dots|c_n\)和\(c_1|\dots|c_n\)不同，即任意一个向量都无法包含在其余向量的或结果中。本题即求给定向量组\(c_1,c_2,\dots,c_n\)是否为或无关。

    代码

    #include <bits/stdc++.h>

using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int t;
    cin >> t;
    vector<vector<int>> bits;
    map<int, int> num;
    while (t--) {
        int n;
        cin >> n;
        bits.resize(n, {});
        num.clear();
        for (int i = 0; i < n; ++i) {
            int k;
            cin >> k;
            bits[i].resize(k);
            for (int j = 0; j < k; ++j) {
                int p;
                cin >> p;
                bits[i][j] = p;
                auto it = num.insert({p, 1});
                if (!it.second)
                    ++it.first->second;
            }
        }

        bool legal = false;
        for (int i = 0; i < n; ++i) {
            bool lonely = false;
            for (auto c : bits[i])
                if (num[c] == 1) {
                    lonely = true;
                    break;
                }
            if (!lonely) {
                legal = true;
                break;
            }
        }
        cout << (legal ? "YES" : "NO") << endl;
    }
}
