    [Project-FFmpeg] FFmpeg常见操作2

    合并视频流和音频流

    给定一个无声视频（Mountains.mp4），一个音频（Summer.mp4）。要把他们合成一个有伴奏的视频。

    基本流程如下：

    1. -i指定输入FFmpeg的文件路径，两个输入文件则-i两次。
    2. 文件经过解复用器(Demuxer)和解码器(Decoder)解码得到原始视频数据，FFmpeg根据文件格式选择解码器。
    3. -shortest表示最短时间的那个流结束时，视频结束。
    4. 指定输出文件格式，让处理后的数据经编码器(Encoder)和复用器(Muxer)编码成视频文件。只有一个输出文件的时候，默认合并所有输出流（不冲突的话）。

    shortest选项介绍如下：

    Option Description
    -shortest (output) Finish encoding when the shortest output stream ends.

    输入命令：

    ffmpeg -i Mountains.mp4 -i Summer.mp3 -shortest Mountain_Summer.mp4

    输出文件（Mountain_Summer.mp4）：

    从视频中提取视频流和音频流

    现在要从视频中提取出无声视频流和音频流，以Mountain_Summer.mp4为例。

    基本流程如下：

    1. -i指定输入FFmpeg的文件路径。
    2. 视频文件经过-map解复用器(Demuxer)选择流，经过解码器(Decoder)解码得到原始视频数据，FFmpeg根据文件格式选择解码器。
    3. 指定输出文件格式，让处理后的数据经编码器(Encoder)和复用器(Muxer)编码成输出文件。

    -map解复用器（流选择器）介绍如下：

    Option Description
    -map 0:a:1 input file 0, audio stream 1
    -map 0:v:0 input file 0, video stream 0

    输入命令： 

    ffmpeg -i Mountain_Summer.mp4 -map 0:a:0 Audio.mp3

    输出文件（Audio.mp3）：

    输入命令： 

    ffmpeg -i Mountain_Summer.mp4 -map 0:v:0 Video.mp4

    输出文件（Video.mp4）：

    在时间轴上裁剪视频

    给定一个视频，裁剪某一段时间的视频流和音频流，以Mountain_Summer.mp4为例。

    基本流程如下：

    1. -i指定输入FFmpeg的文件路径。
    2. 视频文件经过解复用器(Demuxer)和解码器(Decoder)解码得到原始视频数据，FFmpeg根据文件格式选择解码器。
    3. 指定时间裁剪开始处(ss)和持续时间(t)两个过滤器(Filter)和参数，处理原始视频数据。
    4. 指定输出文件格式，让处理后的数据经编码器(Encoder)和复用器(Muxer)编码成视频文件。

    -ss开始时间过滤器，-t持续时间过滤器介绍如下：

    Option Description
    -ss hh:mm::ss Note that in most formats it is not possible to seek exactly, so ffmpeg will seek to the closest seek point before position.
    -t hh:mm::ss duration must be a time duration specification

    输入命令：

    ffmpeg -i Mountain_Summer.mp4 -ss 00:00:05 -t 00:00:03 Clipped_Mountain_Summer.mp4
