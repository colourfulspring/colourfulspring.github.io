    子集遍历

    若u，s的二进制形式表示每个元素是否在全集U，集合S中，那么s-1代表s中最右侧的1变为0，且该1右侧的所有0变为1。考虑s-1中变为1的那些位可能本身不位于全集U中，所以我们要将s-1和u按位与。

    这样达到的效果是：每次将某个元素从集合中删除，并再次将之前删除的所有元素放入集合。一直进行这个操作就能让s的二进制形式取遍集合S的所有子集。

    例如u=101b，则s先后为101b，100b，001b，000b。

    // 遍历 u 的非空子集
for (int s = u; s; s = (s - 1) & u) {
  // s 是 u 的一个非空子集
}

    二进制表示中1的个数

    s-1的二进制形式表示s二进制形式中最右侧的1变为0，且该1右侧的所有0变为1。现在我们需要将s中最右侧的1变为0，从而循环该过程以计算s中1的个数。

    由s-1的性质可知，只要将s与s-1按位与，便可以满足需要。

    例如s=1010b，则s将先后为1010b，1000b，0000b。

    for (int s = n; s; s = s & (s - 1), ++ans);
// ans是n二进制表示中1的个数
